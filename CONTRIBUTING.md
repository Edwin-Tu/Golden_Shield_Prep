# Contributing Guide

歡迎一起協作整理資安學習內容。

## 分支規範
請使用以下命名方式：
- `feature/add-picoctf-writeup`
- `docs/update-nmap-notes`
- `fix/correct-sqli-note`

## Commit 訊息建議
- `docs: add picoCTF web challenge writeup`
- `feat: add burpsuite basic notes`
- `fix: revise xss explanation`

## Pull Request 建議
PR 內容請包含：
1. 本次新增/修改的內容
2. 修改原因
3. 是否需要他人補充或 review

## 文件撰寫建議
- 使用 Markdown
- 儘量包含標題、目錄、步驟、重點整理
- 指令請用 code block
- 工具筆記要寫出用途、基本語法、常見案例
- Writeup 不只寫 flag，應寫思路與學到的東西

## 禁止事項
- 不要提交真實憑證、token、密碼
- 不要提交未授權目標的攻擊紀錄
- 不要只貼 AI 產生內容而不驗證
